#   - mediapipe 0.10.14 is the only release with both linux/aarch64 AND cp312

# with the mediapipe cap above. The pipeline accepts this: edit.js wraps the
# auto-editor call in a try/catch and falls back to the un-trimmed cut, so a
# missing binary is non-fatal — clips still ship, just without silence trim.
# Users who want silence trim can install a working auto-editor themselves
# (e.g. `pip install --no-deps auto-editor==25.3.1` after this base install).
